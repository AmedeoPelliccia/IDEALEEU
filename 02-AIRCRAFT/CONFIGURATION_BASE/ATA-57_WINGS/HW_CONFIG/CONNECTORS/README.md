# Wing Connectors

This directory contains wing-specific connector definitions and documentation.

## Contents

- **WING_ROOT_CONNECTORS.pdf** - Wing root electrical and hydraulic connectors
- **CONTROL_SURFACE_CONNECTORS.pdf** - Aileron, flap, slat, and spoiler connectors
- **SENSOR_CONNECTORS.pdf** - Sensor and instrumentation connectors

## Purpose

Connector documentation provides:
- Connector pinout diagrams
- Pin assignments and signal definitions
- Connector specifications (type, series, manufacturer)
- Mating procedures and torque requirements
- Testing and continuity check procedures

## Wiring Reference

All wiring configurations and harness routing reside in **ATA-92_EWIS** per EWIS rules. This directory only contains connector interface definitions.
