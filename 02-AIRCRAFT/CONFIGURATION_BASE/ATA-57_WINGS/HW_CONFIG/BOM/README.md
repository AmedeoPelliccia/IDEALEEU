# Wing Bills of Materials

This directory contains wing hardware bills of materials (BOMs).

## Contents

- **LEFT_WING_BOM.csv** - Complete BOM for left wing assembly
- **RIGHT_WING_BOM.csv** - Complete BOM for right wing assembly
- **WING_CONTROL_SURFACES_BOM.csv** - BOM for wing control surfaces (ailerons, flaps, slats, spoilers)

## BOM Structure

Each BOM file includes:
- Item number and nomenclature
- Part number (manufacturer and internal)
- Quantity per assembly
- Unit of measure
- Reference designator
- Vendor/supplier information
- Cage code
- Configuration applicability

## Purpose

BOMs provide:
- Complete parts list for procurement
- Manufacturing build requirements
- Configuration control and effectivity tracking
- Spare parts planning
- Maintenance parts ordering
