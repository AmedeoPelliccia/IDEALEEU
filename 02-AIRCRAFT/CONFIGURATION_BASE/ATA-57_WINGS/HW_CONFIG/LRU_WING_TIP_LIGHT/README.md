# LRU: Wing Tip Light

## Overview

Wing tip lights provide navigation and position lighting for aircraft visibility. This directory contains complete documentation for wing tip light LRUs including specifications, installation procedures, maintenance manuals, and component breakdowns.

## Contents

### Top-Level Documents
- **SPECIFICATIONS.pdf** - Technical specifications and performance characteristics
- **PART_NUMBERS.csv** - Part numbers, variants, and interchangeability data
- **INSTALLATION.pdf** - Installation procedures and mounting requirements
- **TEST_PROCEDURES.pdf** - Acceptance testing and functional verification

### ACM/ - Aircraft Component Manuals
- **ACM_WING_TIP_LIGHT.pdf** - Master component manual with complete system description
- **COMPONENT_BREAKDOWN.pdf** - Exploded views and assembly diagrams
- **MAINTENANCE_MANUAL.pdf** - Maintenance tasks, schedules, and procedures
- **ILLUSTRATED_PARTS_CATALOG.pdf** - Parts identification and ordering information
- **TROUBLESHOOTING_GUIDE.pdf** - Fault isolation and diagnostic procedures

### SUBCOMPONENTS/ - Subassembly Documentation
- **LED_ASSEMBLY.pdf** - Light emitting diode array
- **POWER_SUPPLY_MODULE.pdf** - Voltage regulation and conditioning
- **LENS_ASSEMBLY.pdf** - Optical lens and housing
- **MOUNTING_HOUSING.pdf** - Weatherproof enclosure

## Technical Summary

**Function**: Navigation and position lighting
**Type**: LED lighting assembly
**Location**: Wing tips (left and right)
**Interfaces**: ATA-33 Lights, ATA-24 Electrical Power, ATA-92 EWIS
